Painel OnScreen

Entre com sua chave de acesso do GitHub para gerenciar as telas.

Como pegar minha chave? (só na primeira vez)
  1. Abra github.com/settings/personal-access-tokens/new (logado na conta OnScreen).
  2. Em Token name, escreva: Painel OnScreen.
  3. Em Expiration, escolha No expiration (sem validade).
  4. Em Repository access, marque Only select repositories e escolha onmidiascreen-dotcom.github.io.
  5. Em Permissions → Repository permissions, ache Contents e mude para Read and write.
  6. Clique em Generate token, copie o código que aparece e cole aqui em cima.

Guarde essa chave; ela fica salva neste aparelho e você não precisa digitar de novo.
